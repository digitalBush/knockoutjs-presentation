The Goal
  • Define simple javascript models.
  • Write templates.
  • Marry the two together with data binding.
Observable
  • Subscribe with a callback.
  • Get notified when value changes.
Observable Example
//Construction            
var viewModel={
    Name:ko.observable("Josh Bush")
};

//Set Value
ko.Name("digitalBush");

//Get Value
var who = ko.Name();
Observable Array
  • Subscribe with a callback.
  • Get notified when array contents change.
  • Exposes methods from underlying array like push, pop, shift, unshift, etc.
Observable Array Example
//Construction            
var viewModel={
    Tags:ko.observableArray([
        "Javascript",
        "Templating",
        "Binding"
    ])
};

//Get Underlying Array
var items=ko.Tags();
Manipulating Observable Array Items
//New item on the end
viewModel.Tags.push('New Tag');

//Remove the last item
viewModel.Tags.pop();

//New item on the front
viewModel.Tags.unshift('New Tag');

//Remove the front item
viewModel.Tags.shift();
It keeps going...
//Reverse
viewModel.Tags.reverse();

//Sort
viewModel.Tags.sort();
Computed Observable
  • Define a function to compose other observable value(s)
  • Dependent observable subscriptions autowired
  • Subscribe with a callback.
  • Get notified when value changes.
Computed Observable Example
var viewModel={
    Subtotal:ko.observable(5)
};

viewModel.Tax=ko.computed(function(){
    return this.Subtotal()*0.0925;
},viewModel);
You can write to them too!
viewModel.Total=ko.computed({
    read: function () {
        return this.Subtotal() + this.Tax(); 
    },
    write: function (value) {
        this.Subtotal(value/1.0925)
    },
    owner: viewModel
},viewModel);
